## list

List Collections

### Example Usage

<!-- UsageSnippet language="python" operationID="issueTracking.collectionsAll" method="get" path="/issue-tracking/collections" -->
```python
import apideck_unify
from apideck_unify import Apideck
import os


with Apideck(
    consumer_id="test-consumer",
    app_id="dSBdXd2H6Mqwfg0atXHXYcysLJE9qyn1VwBtXHX",
    api_key=os.getenv("APIDECK_API_KEY", ""),
) as apideck:

    res = apideck.issue_tracking.collections.list(raw=False, service_id="salesforce", limit=20, sort={
        "by": apideck_unify.CollectionsSortBy.NAME,
        "direction": apideck_unify.SortDirection.DESC,
    }, pass_through={
        "search": "San Francisco",
    }, fields="id,updated_at")

    while res is not None:
        # Handle items

        res = res.next()

```

    consumer_id="test-consumer",
